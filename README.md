# RiskPilot AI

## Overview

RiskPilot AI is a structured AI governance and risk assessment platform designed to help UK SMEs evaluate the regulatory, operational, and ethical risk exposure of their AI systems.

The platform provides explainable risk scoring and governance recommendations aligned with ISO/IEC 42001 principles and emerging AI regulatory frameworks.

---

## The Problem

Small and mid-sized enterprises are rapidly adopting AI systems without formalised risk classification, governance controls, or documentation processes.

As AI regulation evolves, organisations face increasing exposure to:

- Regulatory scrutiny  
- Legal liability  
- Reputational damage  
- Operational risk  
- Inadequate oversight mechanisms  

Most SMEs lack internal AI governance expertise or structured assessment tools.

---

## The Solution

RiskPilot AI provides:

- Structured AI use-case intake and classification  
- Multi-dimensional risk scoring engine  
- Identification of key governance gaps  
- Governance maturity evaluation  
- Recommended control actions aligned to recognised AI standards  
- Explainable risk outputs (not black-box responses)  

The system is designed to be transparent, auditable, and practical for SME environments.

---

## Target Users

- SME Founders deploying AI tools  
- CTOs and Technical Leads  
- Compliance and Risk Officers  
- AI Product Teams operating in regulated industries  

---

## MVP Scope (Phase 1)

- AI system intake questionnaire  
- Rule-based risk scoring engine  
- Governance risk classification (Low / Medium / High)  
- Control recommendations output  
- Risk results dashboard

## Development

### Prerequisites
- Node.js 18+ (recommended: latest LTS)
- npm (or pnpm)

### Run locally



```bash
First, run the development server:
npm install

## Scripts

- `npm run dev` – Start the development server  
- `npm run build` – Create a production build  
- `npm run start` – Start the production server  
- `npm run lint` – Run lint checks  

```

Open [http://localhost:3000] with your browser to see the result.
